# Student Grade Tracker (Python Programming)

## Project Overview

The **Student Grade Tracker** is a simple Python-based console application that calculates a student’s total marks, average marks, and grade based on marks entered for different subjects.

This project is designed for beginners who are learning Python programming fundamentals and want to practice concepts like:

* Variables
* User Input
* Arithmetic Operations
* Conditional Statements
* Output Formatting

The system takes marks for:

* English
* Science
* Maths
* Islamiat
* Pak Studies

After taking input, the program:

1. Calculates total marks
2. Calculates average marks
3. Displays the final result
4. Assigns a grade based on the average

#  Objectives

The objectives of this project are:

* To practice basic Python programming
* To understand user input handling
* To perform calculations using arithmetic operators
* To implement grading logic using conditions
* To create a mini academic management system
